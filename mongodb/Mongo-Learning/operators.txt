======> For Reading (Filtering):
1. Comparison Operators
$eq: Matches values that are equal to a specified value.
Syntax: { qty: { $eq: 20 } } (or simply { qty: 20 })
$gt / $lt: Matches values that are greater than ($gt) or less than ($lt) a value.
Syntax: { age: { $gt: 18 } } or { age: { $lt: 65 } }
$gte / $lte: Matches values that are greater than or equal to ($gte) or less than or equal to ($lte) a value.
Syntax: { price: { $gte: 100 } } or { price: { $lte: 500 } }
$ne: Matches all values that are not equal to a specified value.
Syntax: { status: { $ne: "active" } }
$in: Matches any of the values specified in an array.
Syntax: { status: { $in: ["A", "D"] } }
$nin: Matches none of the values specified in an array.
Syntax: { grade: { $nin: [20, 30] } }
/*/*/*/*/*/*/*/
2. Logical Operators
$or: Joins query clauses with a logical OR returns all documents that match the conditions of either clause.
Syntax: { $or: [ { price: { $lt: 5 } }, { price: { $gt: 100 } } ] }
$and: Joins query clauses with a logical AND returns all documents that match the conditions of both clauses.
Syntax: { $and: [ { price: { $ne: 0 } }, { qty: { $gt: 10 } } ] }
$not: Inverts the effect of a query expression and returns documents that do not match the query expression.
Syntax: { price: { $not: { $gt: 100 } } }
$nor: Joins query clauses with a logical NOR returns all documents that fail to match both clauses.
Syntax: { $nor: [ { price: 1.99 }, { sale: true } ] }
/*/*/*/*/*/*/*/*/
3. Element Operators
$exists: Matches documents that have the specified field.
Syntax: { "phoneNumber": { $exists: true } }
$type: Selects documents if a field is of the specified type (e.g., String, Number).
Syntax: { "zipCode": { $type: "string" } }
/*/*/*/*/*/*/*/*/*/
4. Array Operators
$all: Matches arrays that contain all elements specified in the query.
Syntax: { tags: { $all: ["ssl", "security"] } }
$elemMatch: Selects documents if an element in the array field matches all the specified conditions.
Syntax: { results: { $elemMatch: { product: "xyz", score: { $gte: 8 } } } }
$size: Selects documents if the array field is a specified size.
Syntax: { tags: { $size: 3 } }

1. Field Update Operators
$set: Replaces the value of a field with the specified value.
Syntax: { $set: { status: "active", score: 10 } }
$unset: Deletes a particular field.
Syntax: { $unset: { "email": "" } }
$inc: Increments the value of the field by the specified amount.
Syntax: { $inc: { views: 1 } } (Use negative numbers to decrease)
$rename: Renames a field.
Syntax: { $rename: { "nickname": "alias" } }
$min / $max: Only updates the field if the specified value is less than ($min) or greater than ($max) the existing field value.
Syntax: { $max: { highScore: 500 } } (Only updates if current high score is < 500)
$currentDate: Sets the value of a field to current date, either as a Date or a Timestamp.
Syntax: { $currentDate: { lastLogin: true } }
/*/*/*/*///*/*/**/*/*/*/*/*/
2. Array Update Operators
$push: Adds an item to an array.
Syntax: { $push: { scores: 89 } }
$pop: Removes the first (-1) or last (1) item of an array.
Syntax: { $pop: { scores: 1 } } (Removes the last item)
$pull: Removes all array elements that match a specified query.
Syntax: { $pull: { votes: { $lt: 5 } } } (Removes all votes under 5)
$addToSet: Adds elements to an array only if they do not already exist in the set.
Syntax: { $addToSet: { tags: "gadget" } }
